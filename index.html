CSS box-shadow allow us to create pretty cool shapes that don't even look like shadows. You can add more than one shadow to elements. You just have to seperate each shadow by a comma. It can be funny to create shapes with box-shadow but sometimes it can be a little bit tiring. For example to create Mario character I use 143 shadows and 987 shadows for the pipe. Crazy huh? :)

You can try to draw Mona Lisa at home. =)

NOTE : Works in Chrome, Firefox, Safari, Opera and Internet Explorer 9 latest versions.
